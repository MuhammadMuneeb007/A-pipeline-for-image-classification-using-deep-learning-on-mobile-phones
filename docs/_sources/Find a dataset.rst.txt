Find a dataset
=========================
 


There are many websites from which medical/plants/objects data can be downloaded.

See (Medical) `This Link <https://github.com/sfikas/medical-imaging-datasets>`_.

See (Medical) `This Link <https://www.aylward.org/notes/open-access-medical-image-repositories>`_.

See (Plants) `This Link <https://www.quantitative-plant.org/dataset>`_.

Sample datasets used to demonstrate the working of this project are mentioned in the paper.
